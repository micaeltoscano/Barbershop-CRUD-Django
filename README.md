# Barbershop CRUD System

## Overview
This project is a CRUD (Create, Read, Update, Delete) web application developed for a Database course. It simulates a barbershop management system, allowing the handling of customers, services, and scheduling through a structured and scalable backend built with Django.

## Project Information
- **Course:** Database  
- **Type:** Academic Project  
- **Framework:** Django  

## Features
- Customer registration and management  
- Service management  
- Appointment scheduling  
- Data persistence using a relational database  
- Basic web interface using Django templates  

## Technologies Used
- **Python 3**  
- **Django**  
- **SQLite / PostgreSQL (configurable)**  
- **HTML / CSS (Django Templates)**  
- **Docker (optional setup)**  

## Project Structure
- `barbearia/`: Main Django project configuration  
- `core/`: Core application with main settings and routing  
- `funcoes/`: Business logic and additional functionalities  
- `templates/`: HTML templates for the frontend  
- `scripts/`: Utility and setup scripts  
- `manage.py`: Django management script  
- `requirements.txt`: Project dependencies  
- `docker-compose.yml`: Container orchestration setup  
- `.env.example`: Environment variables template
